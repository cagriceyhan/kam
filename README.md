# KAM 

# Key Features
- **Multilingual Support:** Full support for English and Turkish.
- **Smart Filtering:** Filter by minimum length or required digit presence.
- **LeetSpeak Engine:** Automatic conversion (e.g., `android` -> `4ndr0id`).
- **Cryptographic Support:** Generate plain text, MD5, SHA1, or SHA256.
- **System Integration:** Can be used directly as a `kam` command in Linux.

# Installation
```bash
git clone [https://github.com/USERNAME/kam.git](https://github.com/USERNAME/kam.git)
